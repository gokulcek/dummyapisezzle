Dummyapi
Basic automation framework to automate REST end point in DUMMYAPI

Description
Framework uses mocha,chai and axios library. It uses mochaawsome for reporting. Typescript is used for scripting.Once execution is completed, execution result will be created in Report folder

Installation
use npm pakage manager to install dependencies after cloning/copying the folder.
update the <APP_ID> in RestUtil.ts file

npm intall
Usage
npm test
# runs all the test in test folder using axios
